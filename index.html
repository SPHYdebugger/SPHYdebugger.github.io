Quién soy

He trabajado en proyectos Java con Spring Boot, desarrollando microservicios escalables y robustos, y gestionando APIs RESTful para distintos servicios. Además, poseo experiencia en gestión de bases de datos relacionales (MySQL, PostgreSQL y Oracle) y no relacionales (MongoDB). Tengo conocimientos de PYTHON para la creación de tareas con Airflow. He utilizado para FrontEnd los lenguajes HTML, CSS, JavaScript y TypeScript, empleando el framework ANGULAR.

Para CD/CI he usado tanto GitHub como GitLab, Jenkins, Docker y Kubernetes, lo que me ha permitido automatizar despliegues y mejorar la eficiencia operativa de los equipos, unidos a en la implementación de sistemas distribuidos y la integración de tecnologías en la nube, como AWS (EC2, RDS, S3).

Además de mis habilidades técnicas, poseo habilidades comunicativas y organizativas para la planificación de proyectos y la gestión de tiempos y equipos. Me considero una persona responsable, positiva y buen comunicador. Capaz de llevar a cabo cualquier reto y aprendiendo en cada paso. Mi enfoque es siempre la entrega de productos de alta calidad, manteniendo los plazos y ofreciendo soluciones eficaces a los problemas que surgen durante el desarrollo.
